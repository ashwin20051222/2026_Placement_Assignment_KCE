## Content Details

- `01_Assignment_HW.pdf`
  Hardware assignment question paper.
- `02_Assginment_DS.pdf`
  Data structures assignment question paper.
- `03_Assignment_OS.pdf`
  Operating systems assignment question paper.

- `Ass_1_hw/`
  AVR / ATmega328P assignment programs.
  Contains C source files, generated `.elf` and `.hex` files, `assignment_common.h`, `compile_all.sh`, and `mini_projects/`.

- `Ass_2_ds/`
  Data structures programs in C.
  Contains source files, compiled executable files, and `Output/` screenshots for program results.

- `Ass_3_os/`
  Operating systems and IPC programs in C.
  Contains source files, compiled executable files, `Makefile`, and `Output/` screenshots.

- `Lab_manual/`
  Lab manual source and derived work.
  Contains `Lab_Manual.doc`, `manual_extracted.txt`, reference images, and three parts:

- `Lab_manual/part_a/`
  Data structure lab programs in C.
  Contains 15 source files, generated terminal outputs, a helper script `generate_all_outputs.sh`, and compiled files in `bin/`.

- `Lab_manual/part_b/`
  Linux IPC and system programming lab programs in C.
  Contains pipe, message queue, shared memory, semaphore, thread, configuration, file manager, and networking examples.

- `Lab_manual/part_c/`
  STM32 / Keil lab programs.
  Contains assembly files, C driver programs, shared helper files, `README_KEIL_PART_C.md`, and simulated output images in `Simulated_out/`.

- `Softcopy/`
  Document softcopies.
  Contains PDF and DOCX files for Assignment 2 and Assignment 3.
